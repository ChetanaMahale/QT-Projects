# 🧩 QT-Projects

A collection of **Qt 6 (C++ / QML)** showcase applications — from UI fundamentals to networked real-time apps.

---

## 📦 Projects

| # | Project | Description | Key Technologies |
|---|---|---|---|
| 1 | [🔢 Calculator](./calculator/) | Scientific calculator with history log | QML, C++ backend |

## 🛠️ Tech Stack

- **Qt 6.5+** — Quick, Network, Multimedia, Sql
- **CMake 3.16+** — cross-platform build
- **QML** — declarative UI with animations
- **C++ (Qt Slots/Signals)** — backend logic and data management

---

## 🚀 Getting Started

Each project is self-contained with its own `CMakeLists.txt`.

1. Open a project folder in **Qt Creator**
2. Select your **Desktop Kit** (e.g. `Desktop Qt 6.8.x MSVC2022 64bit`)
3. Click **Configure Project** → **▶ Run**
